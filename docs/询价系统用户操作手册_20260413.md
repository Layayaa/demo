# 询价系统用户操作手册（含演示图预留位）

版本：2026-04-13  
适用系统：企业内部历史询价复用系统  
适用对象：普通用户、管理员

---

## 1. 角色与权限说明

### 1.1 普通用户
- 注册账号、登录系统、修改密码
- 智能查询历史询价数据
- 条件筛选并查看记录详情
- 上传询价文件（部门自动取账号信息）
- 查看已上传文件、查看上传人与询价工程师
- 在可用场景下查看联系方式（当前前端已隐藏拨号/短信按钮）

### 1.2 管理员
- 拥有普通用户全部能力
- 用户管理：添加、编辑、重置密码、删除用户
- 工程师关联管理：查看待关联、查看已关联、取消关联后重新绑定

## 2. 普通用户功能操作

## 2.1 注册与登录

### 2.1.1 用户注册
1. 打开注册页：`/register`
2. 填写：用户名、真实姓名、手机号、部门、密码、确认密码
3. 点击“注册”
4. 注册成功后，系统自动执行工程师匹配，并跳转登录页

说明：
- 用户名需为 3-32 位字母/数字/下划线
- 手机号需为 11 位大陆手机号
- 密码需至少 8 位，且包含字母和数字

【演示图预留位-02】注册页面（含密码强度条）  
建议图片文件名：`manual_images/02_用户注册.png`

### 2.1.2 用户登录
1. 打开登录页：`/login`
2. 输入手机号和密码
3. 点击“登录”进入系统主页

【演示图预留位-03】登录页面

### 2.1.3 修改密码
1. 登录后点击右上角“修改密码”
2. 输入旧密码、新密码、确认新密码

## 2.2 智能查询（重点）

### 2.2.1 功能目标
- 用自然语言查询历史询价价格
- 查询“谁上传了这份报价文件”
- 查询“谁负责该材料（询价工程师）”
- 支持追问承接上下文（例如“这份报价谁上传的”）

### 2.2.2 常用问法示例
- 价格查询：`德阳商品混凝土C30最近价格`
- 上传人查询：`这份价格谁上传的`、`谁上传了这个报价`
- 工程师查询：`这个材料谁负责`、`资料在哪个工程师手上`
- 追问：先问价格，再问`这份报价谁上传的`

### 2.2.3 查询结果说明
- 会显示材料、规格、价格、地区、报价日期
- 显示来源信息：上传人、询价工程师
- 支持“查看详情（小眼睛）”
- 支持“联系上传人/联系工程师”（联系方式脱敏显示）

### 2.2.4 联系方式展示规则
- 默认展示脱敏手机号（如 `138****1234`）
- 前端已隐藏“一键拨打/发送短信”按钮
- 若未关联账号，会显示“未关联账号/暂无联系方式”提示

## 2.3 条件筛选

### 2.3.1 筛选字段
- 材料名称（支持模糊）
- 规格型号（支持模糊，可输入部分关键字）
- 地区
- 时间范围（近1年/近6个月/近3个月/全部/自定义）

### 2.3.2 使用步骤
1. 进入“条件筛选”页签
2. 输入任意组合筛选条件
3. 点击“开始筛选”
4. 在结果列表中翻页查看全部记录
5. 点击小眼睛查看单条详情

### 2.3.3 当前结果展示重点
- 不再强调“来源文件”字段
- 重点显示“上传人”和“询价工程师”
- 与智能查询结果展示口径保持一致

## 2.4 上传文件

### 2.4.1 上传前准备
- 建议先查询再上传，避免重复询价
- 文件格式支持：`.xlsx`、`.xls`、`.csv`
- 可先下载系统模板再填报

### 2.4.2 上传步骤
1. 进入“上传文件”页签
2. 选择报价有效期
3. 拖拽或点击选择文件
4. 点击“开始上传”

### 2.4.3 上传逻辑说明
- 填报部门自动取当前登录账号部门，无需手填
- 上传人固定取当前登录账号（与Excel中的询价工程师分开）
- 解析失败文件不会计入“上传成功列表”和成功统计
- 支持重复数据识别与提示

## 3. 管理员功能操作

## 3.1 用户管理

### 3.1.1 进入方式
- 顶部菜单点击“用户管理”（仅管理员可见）

### 3.1.2 功能说明

### 3.1.3 操作建议
- 先编辑再重置，避免误改
- 删除前确认该账号是否仍被业务引用

## 3.2 工程师关联管理

### 3.2.1 进入方式
- 顶部菜单点击“工程师关联”（仅管理员可见）

### 3.2.2 页面结构
- 待关联列表：来自历史记录中的工程师名称，待绑定账号
- 已关联列表：已完成绑定的映射关系

### 3.2.3 核心操作
1. 在待关联列表选择目标用户，点击“绑定”
2. 在已关联列表点击“取消关联”
3. 取消后可重新绑定到其他用户

### 3.2.4 业务影响
- 绑定后：查询结果可展示对应账号的联系方式（脱敏）
- 取消后：对应历史记录中的关联会解除，可重新建立新关联

## 6. 常见问题（FAQ）

1. 问：为什么看不到拨号和短信按钮？  
答：前端已按当前要求临时隐藏拨号/短信功能，仍可查看脱敏联系方式。

2. 问：上传人和询价工程师为什么不同？  
答：上传人来自登录账号；询价工程师来自Excel字段，二者是不同业务角色。

3. 问：智能查询“这份报价谁上传的”为什么有时提示补充关键词？  
答：当上下文不足时，需要补充材料名、规格或文件关键词，系统才可精准定位。

4. 问：取消工程师关联后会怎样？  
答：该关联记录被移除，相关历史记录会解除绑定，可重新关联到正确账号。
