# 企业内部历史询价复用系统 - 启动与操作指南

## 方式一：使用启动脚本（推荐）

双击运行 `start.bat` 文件，脚本会自动：
1. 检查Python环境
2. 安装所需依赖
3. 启动系统

## 方式二：手动启动

### 步骤1：安装依赖

```bash
cd inquiry-system
pip install -r requirements.txt
```

如果下载速度慢，可以使用国内镜像：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 步骤2：启动系统

```bash
cd backend
python app.py
```

### 步骤3：访问系统

在浏览器中打开：http://localhost:5000

## 首次使用流程

### 1. 下载模板

在"文件上传"页面点击"下载Excel模板"获取标准模板。

如果下载失败，可以直接使用：
- `sample_data/询价模板_示例数据.csv`（CSV格式）
- 或手动创建Excel文件，参考 `docs/Excel模板字段说明.md`

### 2. 填写数据

按模板格式填写询价数据，确保：
- 必填字段完整（材料名称、规格型号、单价、地区、报价时间）
- 填报部门和工程师信息准确
- 日期格式统一（建议：YYYY-MM-DD）

### 3. 上传文件

在"文件上传"页面：
1. 填写填报部门、填报工程师
2. 选择准备好的Excel文件
3. 点击"开始上传"

### 4. 查看结果

上传成功后：
- 系统会显示解析的记录数
- 如有重复询价，会显示黄色警告框
- 可在"历史查询"页面检索记录
- 可在"统计分析"页面查看统计数据

## 常见问题

### Q1：启动时提示"ModuleNotFoundError"

原因：依赖未安装或安装不完整

解决：
```bash
pip install flask flask-cors flask-sqlalchemy pandas openpyxl
```

### Q2：启动时提示"端口5000被占用"

原因：5000端口已被其他程序使用

解决：
修改 `backend/app.py` 最后一行：
```python
app.run(debug=True, host='0.0.0.0', port=5001)  # 改为5001或其他端口
```

### Q3：上传时提示"文件解析失败"

原因：Excel格式不符合要求

解决：
1. 确保文件是 .xlsx 或 .xls 格式
2. 确保列名正确（参考模板字段说明）
3. 确保必填字段不为空

### Q4：数据库文件在哪里？

位置：`inquiry-system/database/inquiry.db`

这是一个SQLite数据库文件，可以直接备份或迁移。

### Q5：如何清空数据重新开始？

删除数据库文件后重新启动系统：
```bash
rm database/inquiry.db
python backend/app.py
```

系统会自动创建新的数据库。
