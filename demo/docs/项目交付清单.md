## 四、启动验证

### 启动步骤

1. 进入项目目录：
   ```bash
   cd d:/agent/hello-agents/inquiry-system
   ```

2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

3. 启动系统：
   ```bash
   cd backend
   python app.py
   ```

4. 访问系统：
   - 浏览器打开：http://localhost:5000
