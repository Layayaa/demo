# 企业内部历史询价复用系统

## 项目简介

本系统实现企业内部历史询价数据的复用，遵循"先查后询 → 询完必录 → 录后可溯 → 溯后可用"的业务闭环，帮助企业避免重复询价、提高采购效率、实现价格数据的价值沉淀。

**核心功能**：
- 🔍 自然语言查询 - 智能解析查询意图
- 📊 询价数据管理 - 上传、解析、存储
- 👥 用户权限管理 - 登录认证、角色权限
- 📈 统计分析 - 数据可视化、去重统计

## 技术架构

### 开发环境
```
前端：Bootstrap 5 + 原生 JavaScript
后端：Flask + Flask-Login + SQLAlchemy
数据库：SQLite（四库分离）
├── inquiry_file.db  - 文件管理
├── price_record.db  - 询价明细
├── upload_audit.db  - 审计留痕
└── user.db          - 用户管理
```

### 生产环境（推荐）
```
前端：Bootstrap 5 + 原生 JavaScript
后端：Flask + Gunicorn + Nginx
数据库：MySQL 8.0
进程管理：Supervisor
```

## 功能演示

### 一、用户登录

系统启动后访问 http://localhost:5000 会自动跳转到登录页。

**默认管理员账号**：
- 手机号：`13800138000`
- 密码：`138000`（手机号后六位）

### 二、自然语言查询

在查询框输入自然语言，系统自动解析：

| 输入示例 | 解析结果 |
|---------|---------|
| 成都最近一个月钢筋C30的价格 | 材料:钢筋, 规格:C30, 地区:成都, 时间:最近1个月 |
| 门的报价 | 材料:门, 时间:近1年(默认) |
| 上个月钢材趋势 | 材料:钢材, 时间:上个月, 意图:趋势分析 |

### 三、询价数据上传

1. 填写填报部门、工程师信息
2. 上传 Excel 文件（.xlsx/.xls）
3. 系统自动解析并入库
4. 支持重复数据检测

### 四、用户管理（管理员）

- 添加/编辑/删除用户
- 重置用户密码
- 禁用/启用账号
